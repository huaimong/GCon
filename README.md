### 编译 Build
用Microsoft Visual Studio打开V2RayGCon.sln  
左侧解决方案资源管理器中右键点击V2RayGCon,选上"设为启动项目"  
快捷工具栏中的 Debug Any CPU 改为 full Any CPU  
点菜单"生成"-"重新生成解决方案"  
  
编译出来的文件在"V2RayGCon\V2RayGCon\bin\full\"目录中
